    Methodology

    Anti-Leakage Design

    Three layers prevent future information from contaminating predictions.

    1 Shifted Rolling Features

    All rolling stats use .shift(1) so the current game's outcome is never included in its own features. A 10-game average at game N uses games 1 through N-1.

    2 Prior-Season Joins

    Team quality stats (SRS, Off Rtg, Def Rtg) are joined from season N-1 to predict season N games. No in-season aggregate team stats leak into game-level predictions.

    3 Temporal Split

    Train on 2001-2021, tune on 2022-2023, test on 2024-2025. No future season data ever touches training. Live 2026 season used for active predictions only.
