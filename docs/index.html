Methodology

Anti-Leakage Design

Three layers prevent future information from contaminating predictions.

1 Shifted Rolling Features

All rolling stats use .shift(1) so the current game's outcome is never included in its own features.

2 Prior-Season Joins

Team quality stats (SRS, Off Rtg, Def Rtg) joined from season N-1 to predict season N. No in-season stats leak.

3 Temporal Split

Train 2001-2021, tune 2022-2023, test 2024-2025. No future data ever touches training.
